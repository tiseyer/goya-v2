## Overview

Teachers, wellness practitioners, and school owners can submit events to the GOYA community calendar. Accepted event types include workshops, teacher trainings, dharma talks, conferences, yoga sequences, music playlists, and research events.

All events are reviewed by the GOYA team before they go live. Go to **Settings > My Events** to manage your submissions.

---

## Status flow

Every event moves through the following statuses:

```
[Draft] --> [Pending Review] --> [Published]
                |
                v
           [Rejected] --> (edit) --> [Pending Review]
```

You control the transition from **Draft** to **Pending Review** by clicking **Submit for Review**. The GOYA team handles the transition to **Published** or **Rejected**.

---

## Creating a new event

1. Go to **Settings > My Events**.
2. Click **+ Add New Event**.
3. The first time you create an event, a short information modal appears explaining the submission process. Click **Got it, create my event** to continue.
4. Fill in the event form (see fields below).
5. Click **Save as Draft** to save your work without submitting, or **Submit for Review** to submit immediately.

---

## Event form fields

| Field | Required | Notes |
|---|---|---|
| **Title** | Yes | The name of your event |
| **Category** | Yes | Workshop, Teacher Training, Dharma Talk, Conference, Yoga Sequence, Music Playlist, or Research |
| **Format** | Yes | Online, In Person, or Hybrid |
| **Start Date** | Yes | The date the event begins |
| **End Date** | No | For multi-day events; must be on or after the start date |
| **All Day Event** | No | Check this to hide the start and end time fields |
| **Start Time** | Yes (unless All Day) | Event start time |
| **End Time** | Yes (unless All Day) | Event end time |
| **Instructor** | No | Name of the instructor if different from you |
| **Location** | No | Physical address or online platform (e.g. "Online via Zoom" or "Berlin, Germany") |
| **Description** | No | Full description of the event |
| **Registration Required** | No | Toggle on to reveal pricing and capacity fields |
| **Price** | No | Shown when registration is required. Enter a price, or tick **This event is free** |
| **Total Spots** | No | Shown when registration is required. Leave empty for unlimited capacity |
| **Spots Remaining** | No | Shown when registration is required. Current availability |
| **Event Website** | No | Link to an external event page or registration site |
| **Featured Image** | No | JPEG, PNG, or WebP, max 5 MB |

The form requires **Title**, **Start Date**, and (unless **All Day Event** is checked) **Start Time** and **End Time** before the save or submit buttons become active.

## Saving versus submitting

- **Save as Draft** — saves your event in **Draft** status. You can return to edit it at any time before submitting. Drafts are not visible to the public.
- **Submit for Review** — saves and submits your event. It moves to **Pending Review** and enters the GOYA review queue. You cannot edit it while it is under review.

---

## Status reference and available actions

### Draft

Your event is saved but not yet submitted. It is not visible to anyone else.

**Available actions:**
- **Edit** — open the form to make changes
- **Submit for Review** — send the event to the GOYA team for review
- **Delete** (trash icon) — permanently remove the event

---

### Pending Review

Your event has been submitted and is waiting for the GOYA team to review it. This typically takes a few business days.

**Available actions:**
- **Delete** (trash icon) — withdraw the submission and remove the event

You cannot edit an event while it is in **Pending Review**. If you need to make changes, delete it and create a new submission.

---

### Published

Your event has been approved and is visible on the GOYA community calendar.

**Available actions:**
- **View** — opens your published event page in a new view
- **Delete** (trash icon) — removes the event from the calendar

---

### Rejected

Your event was not approved. A **Rejection reason** is shown beneath the event listing in a red highlighted box — read this carefully before revising your submission.

**Available actions:**
- **Edit** — open the form to update your event based on the feedback
- **Delete** (trash icon) — remove the event entirely

After editing, click **Resubmit for Review** (the submit button relabels automatically for rejected events). Your event returns to **Pending Review**.

---

## Deleting an event

Clicking the trash icon next to any event shows a confirmation prompt. Click **Confirm** to permanently delete the event, or **Cancel** to go back. Deletion cannot be undone.
