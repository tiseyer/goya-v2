# Requirements: Analytics & Tracking System

**Defined:** 2026-04-01
**Core Value:** Data-driven visibility into platform health.

## v1 Requirements

### Navigation
- [ ] **NAV-01**: Admin analytics tabs ordered: Visitors → Users → Shop

### Users Analytics
- [ ] **USER-01**: Stat cards showing Real Members, Teachers, Students, WPs, Schools, Guests, Fake/Test counts
- [ ] **USER-02**: Fake users excluded via wp_roles contains 'faux' OR 'robot'
- [ ] **USER-03**: Member growth area chart with cumulative totals over time
- [ ] **USER-04**: Time filter pills (30D, 90D, 6M, 1Y, YTD, All) default 30D
- [ ] **USER-05**: Role filter pills (All, Teachers, Students, Wellness, Schools) default All
- [ ] **USER-06**: Recent signups table (10 latest real users with avatar, name, email, role, date, admin link)

### Visitors Analytics
- [ ] **VIS-01**: GA4 Data API client in lib/analytics/ga4.ts with service account auth
- [ ] **VIS-02**: Top stats row: Sessions, Pageviews, Unique Users, Bounce Rate, Avg Session Duration, New Users
- [ ] **VIS-03**: Trend indicators vs previous period (up/down arrow + percentage)
- [ ] **VIS-04**: Time filter pills (7D, 30D, 90D, 6M) default 30D
- [ ] **VIS-05**: Traffic chart (sessions over time, recharts)
- [ ] **VIS-06**: Top Pages table (path, sessions, pageviews, avg time)
- [ ] **VIS-07**: Traffic Sources table (source/medium, sessions, % total)
- [ ] **VIS-08**: Countries table (country, sessions, % total)
- [ ] **VIS-09**: Devices breakdown (desktop/mobile/tablet)
- [ ] **VIS-10**: Graceful fallback when GA4 not configured

### Event Tracking
- [ ] **EVT-01**: Tracking utility at lib/analytics/events.ts with predefined events
- [ ] **EVT-02**: Auth tracking (sign_up, login with method)
- [ ] **EVT-03**: Onboarding tracking (started, completed with role)
- [ ] **EVT-04**: Shop tracking (add_to_cart, begin_checkout, purchase)
- [ ] **EVT-05**: School tracking (registration started, completed)
- [ ] **EVT-06**: Content tracking (event/course viewed, enrolled, completed)
- [ ] **EVT-07**: Profile tracking (updated, avatar uploaded)
- [ ] **EVT-08**: Connection tracking (requested, accepted)
- [ ] **EVT-09**: Search tracking (query, results count)

### Documentation
- [ ] **DOC-01**: Manual GA4 setup guide at docs/analytics-manual-setup.md
