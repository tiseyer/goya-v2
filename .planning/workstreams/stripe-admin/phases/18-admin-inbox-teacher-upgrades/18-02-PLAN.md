**Sub-tab definitions:**
```typescript
const SUB_TABS: Array<{ key: 'pending' | 'approved' | 'rejected'; label: string }> = [
  { key: 'pending', label: 'Pending' },
  { key: 'approved', label: 'Approved' },
  { key: 'rejected', label: 'Rejected' },
]
```

**handleApprove(requestId: string):**
1. `setBusy(requestId)`
2. Call `approveUpgradeRequest(requestId)`
3. On success: update the request in local state to `status: 'approved'`
4. `setBusy(null)`
5. On error: `alert(result.error)` (simple feedback, consistent with admin pattern)

**handleReject(requestId: string):**
1. `setBusy(requestId)`
2. Call `rejectUpgradeRequest(requestId, rejectReason)`
3. On success: update request in local state to `status: 'rejected', rejection_reason: rejectReason`, then `setRejectOpen(null)`, `setRejectReason('')`
4. `setBusy(null)`

**Filtered requests:** `const filtered = requests.filter(r => r.status === activeSubTab)`
