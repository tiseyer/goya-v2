Create `app/admin/inbox/actions.ts` as a `'use server'` file.

Export two async functions:

---

### approveUpgradeRequest(requestId: string): Promise<{ success: boolean; error?: string }>

1. **Auth guard:** Use `createSupabaseServerActionClient` from `@/lib/supabaseServer` to get the current user. Fetch their role from `profiles`. If role is not `'admin'`, return `{ success: false, error: 'Unauthorized' }`.

2. **Load request:** Use `getSupabaseService()` to fetch the upgrade_request by `requestId`. Select: `id, user_id, stripe_payment_intent_id, status`. If not found or status is not `'pending'`, return `{ success: false, error: 'Request not found or already reviewed' }`.

3. **Load profile:** Fetch `profiles` for `request.user_id`. Select: `stripe_customer_id, full_name, email`. The profile must have a `stripe_customer_id` — if null, return `{ success: false, error: 'User has no Stripe customer ID' }`.

4. **Capture payment:** Call `getStripe().paymentIntents.capture(request.stripe_payment_intent_id)`. If this throws, catch and return `{ success: false, error: err.message }`.

5. **Create subscription:** Call:
   ```typescript
   const subscription = await getStripe().subscriptions.create({
     customer: profile.stripe_customer_id,
     items: [{ price: 'price_1TE4kfDLfij4i9P9sUpSD2Si' }],
     metadata: { goya_user_id: request.user_id },
   })
   ```
   If this throws, catch and return `{ success: false, error: err.message }`.

6. **Update role:** `getSupabaseService().from('profiles').update({ role: 'teacher' }).eq('id', request.user_id)`

7. **Update request record:**
   ```typescript
   getSupabaseService().from('upgrade_requests').update({
     status: 'approved',
     reviewed_at: new Date().toISOString(),
     reviewed_by: adminUser.id,
     stripe_subscription_id: subscription.id,
   }).eq('id', requestId)
   ```

8. **Notify user:**
   ```typescript
   await (supabase as any).from('notifications').insert({
     user_id: request.user_id,
     type: 'teacher_upgrade_approved',
     title: 'Your upgrade was approved!',
     body: 'Congratulations! Your teacher upgrade has been approved. Your Teacher Membership is now active.',
     link: '/settings/subscriptions',
     actor_id: adminUser.id,
   })
   ```

9. **Revalidate:** `revalidatePath('/admin/inbox')`

10. Return `{ success: true }`.

---

### rejectUpgradeRequest(requestId: string, rejectionReason: string): Promise<{ success: boolean; error?: string }>

1. **Auth guard:** Same as above — require admin role.

2. **Load request:** Fetch upgrade_request by `requestId`. Select: `id, user_id, stripe_payment_intent_id, status`. If not found or status not `'pending'`, return error.

3. **Cancel payment intent:** Call `getStripe().paymentIntents.cancel(request.stripe_payment_intent_id)`. If throws, catch and return error. (This releases the authorization — no charge to user.)

4. **Update request record:**
   ```typescript
   getSupabaseService().from('upgrade_requests').update({
     status: 'rejected',
     rejection_reason: rejectionReason.trim() || null,
     reviewed_at: new Date().toISOString(),
     reviewed_by: adminUser.id,
   }).eq('id', requestId)
   ```

5. **Notify user:**
   ```typescript
   await (supabase as any).from('notifications').insert({
     user_id: request.user_id,
     type: 'teacher_upgrade_rejected',
     title: 'Your upgrade request was not approved',
     body: rejectionReason.trim()
       ? `Your teacher upgrade request was not approved: ${rejectionReason.trim()}`
       : 'Your teacher upgrade request was not approved. Please contact support for details.',
     link: '/settings/subscriptions',
     actor_id: adminUser.id,
   })
   ```

6. **Revalidate:** `revalidatePath('/admin/inbox')`

7. Return `{ success: true }`.
