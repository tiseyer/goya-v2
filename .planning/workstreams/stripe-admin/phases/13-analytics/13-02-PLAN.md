       - `supabase.from('stripe_prices').select('stripe_id, unit_amount, type, interval, interval_count')`
       - `supabase.from('schools').select('owner_id')` — to identify school users

    e. Build schoolOwnerIds: `new Set(schools.map(s => s.owner_id))`
    f. Build profileMap: `new Map(profiles.map(p => [p.id, p]))`
