      ```
      await getSupabaseService().from('stripe_coupon_redemptions').insert({

        user_id: userId,
        stripe_order_id: null,

      })
      ```
    - Increment times_redeemed on stripe_coupons:
      ```
      const { data: coupon } = await getSupabaseService().from('stripe_coupons').select('times_redeemed').eq('stripe_coupon_id', stripeCouponId).single()
      await getSupabaseService().from('stripe_coupons').update({ times_redeemed: (coupon?.times_redeemed ?? 0) + 1 }).eq('stripe_coupon_id', stripeCouponId)
      ```
