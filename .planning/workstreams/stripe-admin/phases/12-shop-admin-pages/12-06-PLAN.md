    - INSERT into stripe_coupon_redemptions (append-only, never update):
      ```
      await getSupabaseService().from('stripe_coupon_redemptions').insert({
        stripe_coupon_id: stripeCouponId,
        user_id: userId,
        stripe_order_id: null,
        redeemed_at: new Date().toISOString(),
      })
      ```
    - Increment times_redeemed on stripe_coupons:
      ```
      const { data: coupon } = await getSupabaseService().from('stripe_coupons').select('times_redeemed').eq('stripe_coupon_id', stripeCouponId).single()
      await getSupabaseService().from('stripe_coupons').update({ times_redeemed: (coupon?.times_redeemed ?? 0) + 1 }).eq('stripe_coupon_id', stripeCouponId)
      ```

    - return `{ success: true }`

    2. Create `app/admin/shop/coupons/[id]/page.tsx` as a Server Component:
    - Accept `{ params: Promise<{ id: string }> }`
    - `const { id } = await params`
    - Handle id === 'new': render CouponForm in create mode with empty fields
    - For existing coupon:
      - Fetch coupon: `supabase.from('stripe_coupons').select('*').eq('id', id).single()`
      - Fetch redemption history: `supabase.from('stripe_coupon_redemptions').select('id, user_id, stripe_order_id, redeemed_at').eq('stripe_coupon_id', coupon.stripe_coupon_id).order('redeemed_at', { ascending: false })`
      - Fetch user profiles for redemption history: collect user_ids, query `supabase.from('profiles').select('id, full_name, email').in('id', userIds)`
      - Fetch order references for redemptions: collect stripe_order_ids (non-null), query `supabase.from('stripe_orders').select('id, stripe_id, amount_total').in('stripe_id', orderIds)`
      - Merge redemptions + profiles + orders
    - Fetch all profiles for assignment dropdown (first 200): `supabase.from('profiles').select('id, full_name, email').order('full_name').limit(200)`
    - Fetch all products for product restrictions multi-select: `supabase.from('products').select('id, name, slug').order('name')`
