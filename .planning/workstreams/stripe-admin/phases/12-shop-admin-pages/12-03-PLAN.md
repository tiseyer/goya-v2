    1. Create `app/admin/shop/orders/actions.ts` with 'use server' + import 'server-only':

    ```
    bulkOrderAction(orderIds: string[], action: 'archive' | 'restore')
    ```
    - For 'archive': `getSupabaseService().from('stripe_orders').update({ status: 'archived' }).in('id', orderIds)`
    - For 'restore': `getSupabaseService().from('stripe_orders').update({ status: 'active' }).in('id', orderIds)`
    - `revalidatePath('/admin/shop/orders')`

    2. Create `app/admin/shop/orders/page.tsx` as a Server Component:
    - Accept `searchParams: Promise<{ [key: string]: string | string[] | undefined }>`
    - Parse filters: `search` (string), `type` (all/one_time/recurring), `status` (all/active/succeeded/failed/canceled/archived), `dateFrom`, `dateTo`, `priceMin`, `priceMax`, `sort` (newest/oldest/amount_high/amount_low), `page`, `pageSize`
    - Build Supabase query on `stripe_orders`:
      ```
      let query = supabase.from('stripe_orders').select('*', { count: 'exact' })
      if (type && type !== 'all') query = query.eq('type', type)
      if (status && status !== 'all') query = query.eq('status', status)
      if (dateFrom) query = query.gte('created_at', dateFrom)
      if (dateTo) query = query.lte('created_at', dateTo + 'T23:59:59Z')
      if (priceMin) query = query.gte('amount_total', parseInt(priceMin) * 100)
      if (priceMax) query = query.lte('amount_total', parseInt(priceMax) * 100)
      // Sort
      switch (sort) {
        case 'oldest': query = query.order('created_at', { ascending: true }); break;
        case 'amount_high': query = query.order('amount_total', { ascending: false }); break;
        case 'amount_low': query = query.order('amount_total', { ascending: true }); break;
        default: query = query.order('created_at', { ascending: false }); break;
      }
      query = query.range(from, to)
      ```
    - Fetch profiles for customer names: collect unique `stripe_customer_id` values from orders, query `profiles` table: `supabase.from('profiles').select('stripe_customer_id, full_name, email').in('stripe_customer_id', customerIds)`
    - If `search` param exists, filter the merged results in JS by matching `full_name.toLowerCase().includes(search)` or `email.toLowerCase().includes(search)`. (Supabase can't do cross-table search in one query without FK.)
    - Fetch coupons for the coupon column: collect unique coupon IDs from orders' metadata (if present), query `stripe_coupons` table
    - Merge orders + profiles + coupons into `OrderRow[]` and pass to OrdersTable
    - Layout: `<div className="p-6 lg:p-8">`, header "Orders" + count, Suspense-wrapped OrdersFilters, OrdersTable, pagination
