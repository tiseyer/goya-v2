- stripe_id, stripe_product_id, currency, unit_amount, type (one_time|recurring), interval, active, created_at, updated_at

Stripe Price immutability rule (locked decision):
- NEVER call stripe.prices.update() with a new unit_amount
- Flow: stripe.prices.create({ product, unit_amount, currency, recurring? }) -> stripe.prices.update(oldPriceId, { active: false })
