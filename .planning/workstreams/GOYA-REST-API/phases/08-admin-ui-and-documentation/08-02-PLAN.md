<interfaces>
<!-- Response envelope format -->
```typescript
interface ApiResponse<T> {
  success: boolean;
  data: T | null;
  error: { code: string; message: string; details?: Record<string, unknown> } | null;
  meta: { timestamp: string; version: string; pagination?: PaginationMeta } | null;
}

interface PaginationMeta {
  page: number; limit: number; total: number;
  total_pages: number; has_next: boolean; has_prev: boolean;
}
```

<!-- Auth methods -->
- Header: `x-api-key: <key>` (primary)
- Header: `Authorization: Bearer <key>` (fallback)
- Permissions: read, write, admin (admin supersedes all)

<!-- Rate limiting -->
- 100 requests per 60-second window per API key
- Returns 429 with Retry-After header when exceeded
</interfaces>
