**`updateUser(id: string, updates: { role?: UserRole; subscription_status?: SubscriptionStatus; member_type?: MemberType | null })`** — per USER-03:
- Validate that `updates` is not empty (has at least one key with a value)
- Allowlist fields: only `role`, `subscription_status`, `member_type` can be set — reject anything else
- Add `updated_at: new Date().toISOString()` to the update payload
- Query: `supabase.from('profiles').update(payload).eq('id', id).select().single()`
- Return `{ data, error }`

**`getUserCredits(userId: string, pagination: PaginationParams)`** — per USER-04:
- Query: `supabase.from('credit_entries').select('*', { count: 'exact' }).eq('user_id', userId)`
- Apply sort from pagination: `.order(pagination.sort, { ascending: pagination.order === 'asc' })`
- Apply range from `paginationToRange(pagination)`
- Allowed sort fields for credits: `['created_at', 'activity_date', 'amount', 'credit_type', 'status']`
- Return `{ data, count, error }`
- Export `CREDITS_SORT_FIELDS` constant

**`getUserCertifications(userId: string)`** — per USER-05:
- Query: `supabase.from('user_designations').select('*, products(name, full_name, category, slug)').eq('user_id', userId).is('deleted_at', null)`
- Use `as any` cast on supabase client (same as other queries — tables not in generated types)
- Return `{ data, error }`

**`getUserVerifications(userId: string)`** — per USER-06:
- Verification data lives ON the profiles table, so query: `supabase.from('profiles').select('id, verification_status, is_verified, certificate_url, certificate_is_official').eq('id', userId).single()`
- Return `{ data, error }`
