```typescript
import { createApiHandler } from '@/lib/api/handler';
import { successResponse } from '@/lib/api/response';

const handlers = createApiHandler({
  GET: async () => {
    return successResponse({
      status: 'ok',
      version: '1.0.0',
      timestamp: new Date().toISOString(),
    });
  },
});

export const GET = handlers.GET;
```

This endpoint:
- Does NOT call validateApiKey (per AUTH-05: no auth required)
- Does NOT call rateLimit (health checks should always be accessible)
- Uses createApiHandler for consistent error handling (try/catch wrapper)
- Returns the standard { success, data, error, meta } envelope via successResponse
- Data includes status:'ok', version:'1.0.0', and current timestamp
