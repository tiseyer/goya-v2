**Part A: Create migration file `supabase/migrations/20260348_event_registrations.sql`:**

```sql
-- ============================================================
-- Migration: Add event_registrations table
-- ============================================================

CREATE TABLE IF NOT EXISTS public.event_registrations (
  id              uuid          PRIMARY KEY DEFAULT gen_random_uuid(),
  event_id        uuid          NOT NULL REFERENCES public.events(id) ON DELETE CASCADE,
  user_id         uuid          NOT NULL,
  registered_at   timestamptz   DEFAULT now(),
  created_at      timestamptz   DEFAULT now(),
  updated_at      timestamptz   DEFAULT now(),
  UNIQUE(event_id, user_id)
);

CREATE TRIGGER update_event_registrations_updated_at
  BEFORE UPDATE ON public.event_registrations
  FOR EACH ROW EXECUTE FUNCTION public.update_updated_at_column();

-- RLS: enabled with no policies (service-role-only access, same as api_keys)
ALTER TABLE public.event_registrations ENABLE ROW LEVEL SECURITY;

-- Index for fast lookups by event
CREATE INDEX IF NOT EXISTS idx_event_registrations_event_id ON public.event_registrations(event_id);
-- Index for fast lookups by user
CREATE INDEX IF NOT EXISTS idx_event_registrations_user_id ON public.event_registrations(user_id);
```

Note: user_id does NOT have a FK to profiles because profiles may use auth.users as the reference and the relationship may be complex. Keep it as a plain uuid column (same approach as other cross-table references in this codebase).

**Part B: Add two functions to the existing `lib/api/services/events.ts`:**

**`export async function registerUser(eventId: string, userId: string)`** -- per EVNT-06:
- First, check if the event exists and is not deleted: call `getEventById(eventId)`. If no data, return `{ data: null, error: 'EVENT_NOT_FOUND' }`
- Check if already registered: `supabase.from('event_registrations').select('id').eq('event_id', eventId).eq('user_id', userId).maybeSingle()`
  - If data exists, return `{ data: null, error: 'ALREADY_REGISTERED' }`
- If event has spots_remaining !== null and spots_remaining <= 0, return `{ data: null, error: 'NO_SPOTS' }`
- Insert registration: `supabase.from('event_registrations').insert({ event_id: eventId, user_id: userId }).select().single()`
- If insert succeeds AND the event has spots_remaining (not null), decrement: `supabase.from('events').update({ spots_remaining: event.spots_remaining - 1 }).eq('id', eventId)`
- Return `{ data: registrationRow, error: null }`
- Use `as any` cast on supabase client (event_registrations not in generated types)

**`export async function unregisterUser(eventId: string, userId: string)`** -- per EVNT-07:
- Delete the registration: `supabase.from('event_registrations').delete().eq('event_id', eventId).eq('user_id', userId).select().single()`
- If error or no data, return `{ data: null, error: 'NOT_FOUND' }`
- If the event has spots_remaining (fetch event to check), increment spots_remaining: `supabase.from('events').update({ spots_remaining: event.spots_remaining + 1 }).eq('id', eventId)`
- Return `{ data: deletedRow, error: null }`

**POST handler:**
1. Auth + rate limit + requirePermission('write')
2. Extract eventId from URL: split pathname, find 'events' segment, take next segment
3. Validate eventId UUID format (return 400 if invalid)
4. Parse JSON body -- expect `{ user_id: string }`
5. Validate user_id is present and is a valid UUID format. Return 400 if missing or invalid.
6. Call `registerUser(eventId, userId)`
7. Handle service errors:
   - If error === 'EVENT_NOT_FOUND': `errorResponse('NOT_FOUND', 'Event not found', 404)`
   - If error === 'ALREADY_REGISTERED': `errorResponse('CONFLICT', 'User is already registered for this event', 409)`
   - If error === 'NO_SPOTS': `errorResponse('NO_SPOTS', 'No spots remaining for this event', 409)`
   - Other errors: `errorResponse('REGISTRATION_ERROR', 'Failed to register user', 500)`
8. Audit log: `await ctx.logAudit({ category: 'admin', action: 'event.register', target_type: 'event_registration', target_id: eventId, description: 'Registered user for event via API', metadata: { user_id: userId, event_id: eventId } })`
