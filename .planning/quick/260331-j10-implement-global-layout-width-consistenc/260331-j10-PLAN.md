Create a reusable PageContainer component in app/components/ui/PageContainer.tsx that encapsulates the canonical layout width pattern.

```tsx
export default function PageContainer({
  children,
  className,
  as: Tag = 'div',
}: {
  children: React.ReactNode
  className?: string
  as?: 'div' | 'section' | 'main'
}) {
  return (
    <Tag className={`max-w-7xl mx-auto px-4 sm:px-6 lg:px-8 ${className ?? ''}`}>
      {children}
    </Tag>
  )
}
```

This is the single source of truth for page content width. Future pages should use this component instead of hardcoding the Tailwind classes.
