1. Import `getSupabaseService` from `@/lib/supabase/service`

   - If profile fetch fails, proceed silently (audit logging must never break calling code)
3. Then call `logAuditEvent(params)` as before
4. Keep the function signature identical â€” `(params: AuditEventParams): Promise<void>`
5. Wrap the profile fetch in try/catch â€” never let it throw

This means ALL client components that pass actor_id will automatically get actor_name and actor_role populated server-side without any client changes beyond ensuring actor_id is passed.
