2. Only commit when `npx tsc --noEmit` passes with 0 errors.

3. When the milestone is complete and the build is green, merge directly to develop:
   `git checkout develop && git merge feature/[milestone-name] && git push origin develop && git branch -d feature/[milestone-name]`

4. Do not push intermediate states to develop. Never commit broken TypeScript.

Format as a numbered list. Keep commands in inline code blocks. The section should be concise — no extra explanation beyond the rules themselves.
