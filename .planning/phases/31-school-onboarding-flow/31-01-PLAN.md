Create `app/schools/[slug]/onboarding/actions.ts` with `'use server'` directive. Import `createSupabaseServerActionClient` from `@/lib/supabaseServer` and `getSupabaseService` from `@/lib/supabase/service`.

Every action takes `schoolSlug: string` as first param. Every action starts with auth check: get user, query school by slug where owner_id=user.id, return error if not found/not owner.

Helper: `async function getOwnedSchool(slug: string)` â€” returns `{ school, error }`. Uses `createSupabaseServerActionClient`, gets user, selects `id, owner_id, status, onboarding_completed` from schools where slug matches and owner_id=user.id. Returns error if not found or onboarding already completed.

**saveBasicInfo(schoolSlug, data: { name: string, short_bio: string, bio: string, established_year: number })**
- Validate: name >= 3 chars, short_bio <= 250 chars required, bio 1000-5000 chars required, established_year between 1900 and current year
- Update school record with these fields
- Return `{ success: true }` or `{ error: string }`

**saveOnlinePresence(schoolSlug, data: { website?: string, instagram?: string, facebook?: string, tiktok?: string, youtube?: string })**
- Validate: at least one field must have a value (at-least-one validation per ONB-03)
- Update school record with these fields
- Return `{ success: true }` or `{ error: string }`

**saveVideoIntro(schoolSlug, data: { video_platform?: 'youtube' | 'vimeo', video_url?: string })**
- If both empty, that's fine (optional step per ONB-04). If platform set, url required.
- Update school with video_platform and video_url (set both null if clearing)
- Return `{ success: true }` or `{ error: string }`

**saveTeachingInfo(schoolSlug, data: { practice_styles: string[], programs_offered: string[], course_delivery_format: 'in_person' | 'online' | 'hybrid', lineage: string, languages: string[] })**
- Validate: practice_styles max 5, languages max 3, course_delivery_format required
- Update school record
- Return `{ success: true }` or `{ error: string }`

**saveLocation(schoolSlug, data: { location_address: string, location_city: string, location_country: string, location_lat: number, location_lng: number, location_place_id: string })**
- Update school record with all location fields
- Return `{ success: true }` or `{ error: string }`

**uploadDocument(formData: FormData)**
- Extract: schoolSlug, designationId, documentType, file from formData
- Auth check via getOwnedSchool
- Upload file to Supabase Storage: `school-documents/{userId}/{schoolId}/{designationId}/{documentType}_{timestamp}.{ext}`
- Get public URL via `getPublicUrl` (or signed URL since bucket is private â€” use `createSignedUrl` for display)
- Insert row into school_verification_documents: school_id, designation_id, document_type, file_url (storage path), file_name, file_size
- Return `{ success: true, document: { id, file_name, file_url } }` or `{ error: string }`

**deleteDocument(schoolSlug, documentId: string)**
- Auth check via getOwnedSchool
- Fetch document row to get file_url (storage path)
- Delete from storage: `supabase.storage.from('school-documents').remove([path])`
- Delete row from school_verification_documents
- Return `{ success: true }` or `{ error: string }`

**saveFacultyMember(schoolSlug, data: { profile_id: string, position: string })**
- Auth check via getOwnedSchool
- Insert into school_faculty: school_id, profile_id, position, status='active'
- Return `{ success: true }` or `{ error: string }`

**removeFacultyMember(schoolSlug, facultyId: string)**
- Auth check via getOwnedSchool
- Delete from school_faculty where id=facultyId and school_id matches
- Return `{ success: true }` or `{ error: string }`

**inviteFacultyByEmail(schoolSlug, data: { email: string, position: string })**
- Auth check via getOwnedSchool
- Generate invite_token: `crypto.randomUUID()`
- Insert into school_faculty: school_id, invited_email=email, invite_token, position, status='pending'
- Return `{ success: true }` or `{ error: string }`
- NOTE: Actual email sending is deferred to Phase 35 (FAC-01). Just save the record for now.

**submitForReview(schoolSlug)**
- Auth check via getOwnedSchool
- Use getSupabaseService() for admin-level updates
- Update school: `onboarding_completed: true, onboarding_completed_at: new Date().toISOString(), status: 'pending_review'`
- Query all admins: `from('profiles').select('id').in('role', ['admin', 'moderator'])`

- Return `{ success: true }` or `{ error: string }`

Create `app/api/schools/faculty-search/route.ts` with a GET handler.

**GET /api/schools/faculty-search?q=searchterm&school_id=uuid**

1. Auth check: `createSupabaseServerClient` -> getUser, return 401 if not authed
2. Parse query params: `q` (search term, min 2 chars), `school_id` (uuid of the school)
3. Verify caller owns the school: query schools where id=school_id and owner_id=user.id, return 403 if not owner
4. Use `getSupabaseService()` to search profiles: `from('profiles').select('id, full_name, avatar_url').ilike('full_name', '%term%').neq('id', user.id).limit(10)`
5. Also exclude profiles already in school_faculty for this school_id: query existing faculty profile_ids first, then chain `.not('id', 'in', '(existingIds)')` or filter in JS
6. Return JSON: `{ results: [{ id, full_name, avatar_url }] }`

Follow the same pattern as `app/api/schools/check-slug/route.ts` for imports and response format.
