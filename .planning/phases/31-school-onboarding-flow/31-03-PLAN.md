**Step 6 — Location (ONB-06):**
- CONDITIONAL: Only show if `school.course_delivery_format === 'in_person' || school.course_delivery_format === 'hybrid'`. If 'online', skip this step entirely (step navigation should go from 5 directly to 7).
- Load Google Places API: Add `<script src="https://maps.googleapis.com/maps/api/js?key=${process.env.NEXT_PUBLIC_GOOGLE_MAPS_API_KEY}&libraries=places" async>` via useEffect. Check `window.google` before initializing.
- Input field with id for autocomplete. Initialize `google.maps.places.Autocomplete` on mount, restricted to types `['establishment', 'geocode']`.
- On place_changed event, extract: `formatted_address` -> location_address, city from `address_components` (locality type) -> location_city, country from `address_components` (country type) -> location_country, `geometry.location.lat()` -> location_lat, `geometry.location.lng()` -> location_lng, `place_id` -> location_place_id.
- Show selected address in a confirmation box below the input.
- Pre-fill if school already has location_address.
- On Continue: call `saveLocation(school.slug, locationData)`, advance on success.
- Back button goes to step 5.

**Step 7 — Document Upload (ONB-07):**
- For EACH designation in the designations prop, show a card with the designation_type as header.
- Each designation card has 3 upload slots:
  1. Business Registration (document_type='business_registration', required) — PDF/JPG/PNG, max 10MB
  2. Qualification Certificate (document_type='qualification_certificate', required) — PDF/JPG/PNG, max 10MB
  3. Insurance Document (document_type='insurance', optional) — PDF/JPG/PNG, max 10MB
- Each slot: file input (hidden) + styled upload button "Choose File" + file name display.
- On file select: create FormData with fields: schoolSlug, designationId, documentType, file. Call `uploadDocument(formData)`. Show upload progress spinner. On success, display file name with a trash icon to delete.
- Pre-fill from documents prop: match by designation_id + document_type. Show existing files with delete option.
- Delete calls `deleteDocument(school.slug, documentId)`.
- Accepted file types: `.pdf,.jpg,.jpeg,.png`
- Validation: all required documents must be uploaded for each designation before Continue is enabled. Show indicator per slot (green check if uploaded, red asterisk if missing).
- On Continue: advance to step 8 (documents are saved individually on upload, no batch save needed).

**Step 8 — Faculty (ONB-08):**
- Top section: "You are listed as Principal Trainer" with owner's name shown.
- Search input: debounced (500ms) fetch to `/api/schools/faculty-search?q=term&school_id=school.id`. Display results in a dropdown below input. Each result shows avatar (or initials), full_name. Click to select.
- On select: show position input modal/inline field. Enter position (e.g. "Senior Teacher"). Call `saveFacultyMember(school.slug, { profile_id, position })`.
- Invite by email section: email input + position input + "Invite" button. Validates email format. Calls `inviteFacultyByEmail(school.slug, { email, position })`.
- Faculty list: show all current faculty from faculty prop + newly added. Each row: name (or email for invited), position, is_principal_trainer badge, remove button (X). Remove calls `removeFacultyMember(school.slug, facultyId)`. Principal trainer row is not removable.
- Pre-fill from faculty prop.
- This step is optional (can Continue with no faculty added beyond the owner).

**Step 9 — Review & Submit (ONB-09):**
- Read-only summary of all entered data, organized in sections:
  - **Basic Info**: name, short bio, established year
  - **Online Presence**: website, social links (only show filled ones)
  - **Video**: platform + URL (or "None")
  - **Teaching**: practice styles as comma-separated, programs, delivery format, lineage, languages
  - **Location**: full address (or "Online School" if format is online)
  - **Documents**: count per designation "X/Y uploaded"
  - **Faculty**: list of names/emails with positions
- Each section has an "Edit" link that navigates back to that step.
- "Submit for Review" button (prominent, centered). On click: call `submitForReview(school.slug)`.
- After successful submit: replace the entire wizard with a confirmation card:
  - Heading: "Your school has been submitted for review"
  - Body: "Our team will review your school and get back to you. This typically takes up to 1 week."
  - "Go to Dashboard" button linking to /dashboard
