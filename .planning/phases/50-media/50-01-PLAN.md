**1. ProfileVideo.tsx** — `'use client'` component implementing CONT-01.

Props: `{ youtubeIntroUrl: string }` (only rendered when url is truthy, parent handles null check).

Facade pattern:
- Parse YouTube video ID from the URL (handle youtube.com/watch?v=ID, youtu.be/ID, youtube.com/embed/ID formats). If URL contains "vimeo", parse vimeo ID similarly.
- State: `playing` boolean, default false.
- When `playing` is false: render a 16:9 aspect ratio container (`aspect-video`) with:
  - YouTube thumbnail: `https://img.youtube.com/vi/{videoId}/maxresdefault.jpg` as background via `<img>` tag with `object-cover`
  - For Vimeo: use a generic placeholder div with a play icon (oEmbed adds complexity; keep it simple)
  - Centered play button overlay: semi-transparent dark circle with white play triangle SVG
  - `onClick` sets `playing = true`
- When `playing` is true: render an iframe in the same 16:9 container:
  - YouTube: `https://www.youtube.com/embed/{videoId}?autoplay=1`
  - Vimeo: `https://player.vimeo.com/video/{videoId}?autoplay=1`
  - `allow="autoplay; encrypted-media"`, `allowFullScreen`
- Wrap in a section with no heading (video speaks for itself).
- Style: `rounded-xl overflow-hidden` on the container.

**2. ProfileMap.tsx** — `'use client'` component implementing MED-01.

Props: `{ lat: number; lng: number }` (only rendered when visibility.showMap is true, parent handles the check).

Follow the pattern from app/members/MapPanel.tsx but drastically simplified:
- `useRef<HTMLDivElement>` for container, `useRef<mapboxgl.Map>` for map instance.
- `useEffect` on mount: set `mapboxgl.accessToken = process.env.NEXT_PUBLIC_MAPBOX_TOKEN`, create map with:
  - `container: containerRef.current`
  - `style: 'mapbox://styles/mapbox/light-v11'` (check dark mode: `document.documentElement.classList.contains('dark')` -> use `dark-v11`)
  - `center: [lng, lat]`
  - `zoom: 13`
  - `interactive: false` (no zoom/pan/scroll — display only)
  - `attributionControl: false`
- Add compact attribution: `map.addControl(new mapboxgl.AttributionControl({ compact: true }), 'bottom-left')`
- Add a single default marker: `new mapboxgl.Marker().setLngLat([lng, lat]).addTo(map)`
- Cleanup: `map.remove()` on unmount
- Import `'mapbox-gl/dist/mapbox-gl.css'` at top
- If `!process.env.NEXT_PUBLIC_MAPBOX_TOKEN`, return null (no render)
- Container styling: `h-[240px] w-full rounded-xl overflow-hidden` on the outer div

Dark mode observer: add a MutationObserver on `document.documentElement` (class attribute) to toggle map style between light-v11 and dark-v11, same as MapPanel.tsx.
