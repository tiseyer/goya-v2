**Auto-send on mount:**
```typescript

```

**sendOTP function:**
- Sets status to 'sending'
- POSTs to `/api/device-verification/send` (no body needed — cookie is httpOnly and sent automatically)
- On success: set status to 'sent', set cooldown to 60
- On error: set status to 'error', set errorMsg

**Cooldown timer:**
```typescript
useEffect(() => {
  if (cooldown <= 0) return
  const timer = setTimeout(() => setCooldown(c => c - 1), 1000)
  return () => clearTimeout(timer)
}, [cooldown])
```

**Auto-submit when OTP is complete:**
When the InputOTP `onChange` fires and the new value has length 6, immediately call `verifyOTP(value)`.

**verifyOTP function:**
- Sets status to 'verifying'
- POSTs to `/api/device-verification/verify` with body `{ code: otpValue }`
- On `{ success: true }`: router.push('/dashboard')
- On error `code_expired`: set errorMsg "Code expired — sending a new one" and call sendOTP()
- On error `too_many_attempts`: set errorMsg "Too many attempts. Please sign out and try again."
- On error `invalid_code`: set errorMsg with remaining attempts if present, reset otp to ''
- On any other error: set errorMsg "Something went wrong. Please try again."

**Sign-out link:**
```typescript
async function handleSignOut() {
  await supabase.auth.signOut()
  router.push('/sign-in')
}
```
