**File 3: app/components/search/GlobalSearchOverlay.tsx**

`'use client'` component. This is the main overlay combining Portal, AnimatePresence, input, pills, results, and keyboard hints.

Structure (follow FlowPlayerModal portal pattern + SearchModal internals):

State:
- `query` (string), `results` (SearchResult[]), `selectedIdx` (number), `activeCategory` (SearchCategory | 'all')
- `inputRef` (useRef<HTMLInputElement>), `debounceRef` (useRef<ReturnType<typeof setTimeout> | null>)
- Import `useSearch()` for `isOpen` and `close`
- Import `useRouter()` from `next/navigation`

Auto-focus on open (useEffect watching isOpen):
- When isOpen becomes true: `setTimeout(() => inputRef.current?.focus(), 50)`, reset query to '', results to [], selectedIdx to 0, activeCategory to 'all'

Debounced mock search (handleInput callback):
- Set query, clear debounceRef, start new 200ms timeout
- In timeout: filter MOCK_RESULTS by `result.title.toLowerCase().includes(query.trim().toLowerCase())` AND (activeCategory === 'all' OR result.category === activeCategory)
- After setting results, reset selectedIdx to 0

Category change handler:
- Set activeCategory, then immediately re-filter MOCK_RESULTS with current query + new category, reset selectedIdx to 0

Keyboard handler (onKeyDown on input):
- `ArrowDown`: `e.preventDefault()`, increment selectedIdx (clamp to results.length - 1)
- `ArrowUp`: `e.preventDefault()`, decrement selectedIdx (clamp to 0)
- `Enter`: if results[selectedIdx] exists, `router.push(results[selectedIdx].href)` then `close()`
- `Escape`: `close()`

Escape keydown listener on document (useEffect, only when isOpen):
- Add `keydown` listener for Escape key -> close()
- Clean up on isOpen change or unmount

Return: `createPortal(<AnimatePresence>{isOpen && (...)}</AnimatePresence>, document.body)`

Desktop layout (hidden on mobile: `hidden sm:flex`):
- Backdrop: `fixed inset-0 z-[9999] bg-black/40 backdrop-blur-sm` with `onClick={close}`
- Panel container: `fixed inset-0 z-[10000] flex items-start justify-center pt-[10vh] pointer-events-none`

   - Clear button (if query.length > 0): X icon 16px `text-slate-400 hover:text-slate-600` onClick resets query and results
   - Close button: X icon 20px `text-slate-400 hover:text-slate-600` onClick calls close(), `aria-label="Close search"`

   - If query.length > 0 and query.trim().length < 2: "Keep typing..." in `text-sm text-slate-400` centered with `py-12`
